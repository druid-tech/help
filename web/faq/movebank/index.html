How to synchronize data to Movebank？

You can synchronize the collected data to Movebank for further study. Before you start, make sure that you have created a Study on Movebank and have added at least one animal to that Study. For more information on creating a Study and adding an animal, refer to [Movebank]: https://www.movebank.org/cms/movebank-content/manual

.

  1. On the main menu of DEBUT data center, click User Center > My Information.

  2. In the Movebank section, click icon to set your Movebank account.
    Type your Movebank account name in the Username field.

    Click Confirm after you finish.
    Note that DEBUT data center can only synchronize data to one Movebank account at the same time.

  3. Log in to Movebank and navigate to your Study.

  4. In the Live Feeds drop-down list, select Druid GSM.

    Movebank enters live feeds management page.

  5. Click New to add Tags.
    In the Add/Remove Tags window, select the available tags and then click Add all Tags.

    Click Save after you finish.

    Note
    You can add Druid GSM live feed to multiple Studies.

  6. Go back to the Study page.

  7. In the Manage drop-down list, select Deployment Manager.
    If you don’t see Deployment Manager option, please wait for 24 hours and try again.

    Movebank enters Deployment Manager page, which is similar to the example below:

    The Tag ID on this page is the Device SN in DEBUT data center.
    Time of data point on this page is the Collecting time in DEBUT data center.

  8. Hover your mouse over a Tag and then click Add animal.

    Add Animal to Tag window pops up.

  9. Select an animal from the drop-down list.

  10. Fill in the necessary information and then click Save.

The Tag is deployed to the animal.

Note

  • DEBUT data center synchronizes the data at 00:00 (UTC+0) every day.
  • When you set your Movebank username on Debut data center for the first time, Debut data center will synchronize all data to Movebank.
  • When you change your Movebank username on Debut data center, Debut data center will only synchronize the data generated from the day when you make the change.
