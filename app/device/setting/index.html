Changing device setting

  1. In DEBUT App, tap Device.
    App enters device list page.
  2. In the device list, tap the device you need to change the setting.
    App enters device information page.
  3. On the device information page, tap Setting.
    App enters device setting page.
  4. Change the settings as needed.
    You can change the following settings:
    • Environment&GPS Sync
      When enabled, the collection mode and collection interval of environmental data will be consistent with the GPS data collection, and cannot be changed separately.
    • GPS data collecting
      Available options are Interval, Follow transmission and Off.
    • Interval
      Set the collecting interval when GPS data collecting is set to Interval.
    • Environment data collecting
      Available options are Interval, Follow transmission and Off. This field is not available when Environment&GPS Sync is enabled.
    • Interval
      Set the collecting interval when Environment data collecting is set to Interval. This field is not available when Environment&GPS Sync is enabled.
    • Behavior data collecting
      Available options are Interval and Off.
    • Interval
      Set the collecting interval when Behavior data collecting is set to Interval.
    • Cellular
      Available options are Interval and On-time.
      This setting doesn’t apply to MINI Argos and NANO.
  5. Click Save after finish.

For devices with a cellular module, the changed settigns will take effect the next time the device communicates with the server. For devices that do not have a cellular module, the changed settings will take effect after the setting is delivered or the HUB associated with the device communicates with the server.
