How to export data?

  1. On the main menu, click Download > Request to go to the request list page.
    Alternatively, you can click Download > Result to go to the result list page. The request list page lists all the exporting tasks that are being generating and not completed. The result list page lists all the completed exporting tasks.
  2. On the request list page or the result list page, click Data download.
  3. Select one or more devices you need to export data in the device list.
  4. Select the data type, data format and time range.
    Available data types are GPS, ENV (environment data), BHV (behaviour data), BSS (base station data), and ACC (raw data collected by the accelerometer).
    Available data formats are CSV and KML (for GPS data only).
    You can select any time range in months before the current time.
  5. Click Export.
    Data center starts generating data in the background. You can check the result in the result list page. The time required to generate data depends on the data amount.
  6. On the main menu, click Download > Result.
  7. In the result list, locate the data and click the download link to export it to your computer.
    The naming convention of the generated data file is SN-Data type-Data format-Time range.zip. If you do not see the data file in the result list, wait for a longer time and refresh the page.
